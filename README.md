Monitor Ultrasónico – Mockup Android

Este proyecto es una interfaz visual desarrollada con Jetpack Compose que representa el diseño de una aplicación móvil para monitorear un sensor ultrasónico conectado mediante Bluetooth.
No incluye la lógica funcional de conexión, solo una simulación visual (mockup) del diseño final de la app.

Características principales:
-Interfaz moderna con Jetpack Compose.
-Boton de Configuración
-Switch de activacion del LED
-Lectura de distancia simulada (--- cm).
-Indicador de estado del dispositivo (“Desconectado”).

Tecnologías utilizadas:
-Kotlin
-Android Studio
-Jetpack Compose
-Material Design 3

Cómo ejecutar el proyecto:
-Abre Android Studio.
-Clona o descarga este repositorio.
-Abre el proyecto y sincroniza Gradle.
-Ejecuta la app en un emulador o dispositivo Android físico.

Este proyecto no requiere permisos de Bluetooth ni hardware real, ya que es solo una representación visual.


Estructura básica del código:
MainActivity.kt: contiene toda la interfaz visual del mockup.
No se incluyen clases de conexión ni servicios Bluetooth, ya que el propósito es únicamente el diseño UI.
